# Silicon X — Website

Marketing website for **Silicon X**, a Thailand-based digital consultancy and transformation
services company (Data · Automation · AI · ITSM).

*Practical technology. Measurable outcomes.*

## Tech
Static site — plain **HTML + CSS + JavaScript**, fully responsive. No build step.

```
index.html · about.html · services.html · ai.html · approach.html · contact.html
css/styles.css   js/main.js   assets/   sitemap.xml   robots.txt
```

## Preview locally
```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Notes (pre-launch)
- Contact form needs a Formspree/Web3Forms endpoint wired in `contact.html`.
- Case-study figures are illustrative examples — confirm before public launch.
